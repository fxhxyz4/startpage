#

### run localy

```
git clone https://github.com/fxhxyz4/startpage.git
cd startpage
code .
```

#

### install packages

```
npm install
```

#

### development

```
npm run dev
```

### open [localhost:1324](http://localhost:1324)

#

### production

```
npm run build
```
